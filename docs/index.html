    get-breakpoint

    @function get-breakpoint($breakpoint: default, $value: false) { ... }

    Description

    When $value is either min or max, the min/max value is returned from the breakpoint map inside $concrete-breakpoints with key $breakpoint. If $value has any other value, the whole map (with min and max keys) will be returned. If the breakpoint map cannot be found, a warning will be stored in $concrete-warnings.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $breakpoint

    The breakpoint to return.

    		Stringdefault
    $value

    The value of the breakpoint to return. This can be either min, max, false or both.

    		Bool or Stringfalse

    Returns

    Map or Number

    fx-gradient-horizontal

    @mixin fx-gradient-horizontal($start, $end: null) { ... }

    Description

    Creates a horizontal gradient. Both start and end values can be either a color or a list with a color and a start/end position percentage. The default starting position is at 0% and end position at 100%. When the end color is not defined, the starting color will be used with 0% alpha, creating a gradient from a solid color to transparent.

    Examples: fx-gradient-horizontal(red); Creates a red to transparent gradient from positions 0% to 100%.

    fx-gradient-horizontal(red, blue); Creates a red to blue gradient from positions 0% to 100%.

    fx-gradient-horizontal(red 5%, blue 95%); Creates a red to blue gradient from positions 0% to 100%;

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $start

    Starting color/position.

    		Color or List—none
    $end

    Ending color/position.

    		Color or List or Nullnull

    Requires

    • [mixin] _fx-gradient-linear
