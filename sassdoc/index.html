    frame-breakpoint

    @function frame-breakpoint($breakpoint: default, $value: false) { ... }

    Description

    When $value is either min or max, the min/max value is returned from the breakpoint map inside $frame-breakpoints with key $breakpoint. If $value has any other value, the whole map (with min and max keys) will be returned. If the breakpoint map cannot be found, a warning will be stored in $frame-warnings.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $breakpoint

    The breakpoint to return.

    		Stringdefault
    $value

    The value of the breakpoint to return. This can be either min, max, false or both.

    		Bool or Stringfalse

    Returns

    Map or Number

    Requires

    • [function] is-null
    • [function] warn

    Used by

    • [mixin] _frame-display-container
    • [function] media-query-list
